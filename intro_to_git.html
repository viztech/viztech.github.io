Introduction to workflow using Git

git

git is a free open source distributed version control system.

git is a tool to manage multiple versions of a document/project when multiple people working/editing.

It tracks changes to any group of documents, files, or folders over time.

Learn more about Distributed Version Controls Systems here.

install git for mac, windows, or linux

GitHub

    GitHub is a platform for hosting and collaborating on projects.

There are GUI softwares available to use git but we are using the command line. This means we will use Terminal on mac and Command Prompt on windows. See this link for getting started with command line on mac.

This is how we use git:
  • First make a copy of the project in your GitHub acount. In GitHub this is called fork.
  • Download the project to your local machine. In GitHub this is called creating a local clone of the repository.
